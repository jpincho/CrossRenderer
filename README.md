# CrossRenderer

An attempt to create a cross-platform cross-graphics api framework

Currently supported platforms:
- Windows ( through GLFW3 )
- Linux ( through GLFW3 )

Currently supported graphics APIs:
- OpenGL Core 3+
	- 2D textures
	- Cubemap textures
	- GLSL shaders
	- Multitexturing
	- Framebuffers
	- Uniform buffer objects
